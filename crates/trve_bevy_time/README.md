# Trve Bevy Time

> Time related utilities.

## Usage

Use `EntityTimer<T>` as a component when spawning your entities or `GlobalTimer<T>` as a resource for your systems.

## Bevy compatibility

| `bevy` Version | `trve_bevy_time` Version |
| -------------- | ------------------------ |
| 0.19           | 0.1, 0.2                 |
